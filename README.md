A real-time, collaborative doodle and drawing application built to demonstrate bidirectional communication between multiple users.

=> Key Features
Real-time Collaboration: Multiple users can draw on the same canvas simultaneously.

Instant Sync: Leverages Socket.io to broadcast brush strokes across all connected clients instantly.

Lightweight Architecture: Powered by Node.js on the backend and vanilla JavaScript/React on the frontend.

=>Tech Stack
Frontend: HTML5 Canvas, CSS3, JavaScript.

Backend: Node.js, Express.

Communication: Socket.io for WebSockets.
